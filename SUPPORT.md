# Getting help

- **Found a bug?** Open a GitHub issue with steps to reproduce.
- **Have a question?** If Discussions are enabled on the repository, start one there. Otherwise, open an issue and label it as a question.
- **Security concern?** Please see [SECURITY.md](./SECURITY.md) and email privately rather than posting publicly.

This is a community project, maintained in spare time. There are no service-level guarantees, but issues are read and usually answered within a reasonable window.
